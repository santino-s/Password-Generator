# Password-Generator

A sleek, web-based tool that generates a random password.

## Features
1. Customizable: Use checkboxes to choose character types and a range or "random" button to choose the password length.
2. Instant Results: Generates a password instantly using JavaScript.
3. Responsive Design: Works on both desktops and mobile devices.

## How to Use
1. Download the files or clone the repository.
2. Open 'index.html' in any web browser.
3. Select your checkboxes and length to generate your password.

## Technologies Used
1. HTML5: For the structure.
2. CSS3: For the styling and layout.
3. JavaScript: For the password generation logic.
